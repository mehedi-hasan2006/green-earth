## 📖 About The Project

**Green Earth** is a web-based initiative designed to educate and inspire people to adopt eco-friendly habits. The project focuses on spreading awareness about climate change, conservation, and sustainable development through a simple and engaging interface.

---

## ✨ Key Features

- 🌿 Clean and simple UI for better user experience  
- 🌎 Awareness about environmental issues  
- ♻️ Tips for sustainable living  
- 🌳 Encouragement for tree plantation  
- 📱 Fully responsive design
